# Pitch For Profit
Pitch For Profit is a MERN stack web application designed to connect entrepreneurs and investors. Entrepreneurs can submit startup pitches, and investors can review these pitches and provide structured feedback to help startups grow and succeed.

## Features

- **Entrepreneur Pitch Submission:** Entrepreneurs can create and submit detailed startup pitches.
- **Investor Feedback:** Investors can browse submitted pitches and provide constructive feedback.
- **User Authentication:** Secure login and registration system for entrepreneurs and investors.
- **Role-Based Access:** Different views and functionalities depending on user roles.
- **Responsive Design:** Works seamlessly on desktop and mobile devices.

## Technologies Used

- **Frontend:** React.js, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **Other:** Axios for API requests
---
Usage
Entrepreneurs can register and submit their startup pitches.

Investors can register, browse pitches, and leave feedback.

Users can log in to access their dashboard and manage pitches/feedback.
